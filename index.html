Projects

Women Diversiform

A secure, full-stack web platform designed to help women report harassment or abuse and access legal and police assistance. Built with React.js, Node.js, and MongoDB, the application includes features such as phone-based authentication, OTP verification, encrypted data handling, complaint tracking, and real-time notifications. The platform prioritizes both user safety and system security, ensuring sensitive data is well protected.

Twitter Recommendation System

Developed a machine learning-based recommendation engine using collaborative filtering to suggest personalized follow recommendations. The system analyzes user behavior, mutual connections, and interaction patterns to generate relevant suggestions. Built in Python, the project focuses on enhancing user engagement and was evaluated for accuracy and user relevance.

User Behavior Prediction

Designed and implemented a logistic regression model to predict whether a user would click on an advertisement, based on behavioral and demographic data. The project involved data preprocessing, model training, and evaluation using metrics like accuracy, precision, and confusion matrix. It provides insights into ad targeting and user engagement strategies.

Address Book Application

Built a responsive contact management application using the MERN stack (MongoDB, Express.js, React.js, Node.js). The app allows users to perform CRUD operations, validate form inputs, and manage contact data efficiently. It features a clean UI and is integrated with RESTful APIs for smooth frontend-backend communication.
